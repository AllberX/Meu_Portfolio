Software por definição:

Software é basicamente um sistema operacional e todos os seus programas associados.

Aplicativo por definição:

            
Aplicação de software, também conhecido como aplicação ou aplicativo, 
é o software que é projetado para ajudar o usuário a executar várias tarefas no computador.

Programa por definição:

 
Um programa (relacionado a conteúdo informático) é um conjunto de instruções que descrevem uma tarefa 
a ser realizada por um computador. Uma referência é o código fonte conhecido por nós da área de T.I, 
que é literalmente escrito em alguma linguagem de programação.

E qual a diferença entre eles ?

 
Confesso pra vocês que considero uma maior concordância e relação entre as definições 
do que uma eventual “diferença”
Mas vamos aos exemplos:
Um Programa = É um conjunto de instruções que definem tarefas a serem executadas sequencialmente 
por QUALQUER tipo de computador.
Aplicações são = Alguns softwares programados para executar varias funções, 
totalmente focado em tarefas muito especificas.
Um Software = É um conjunto de instruções / dados que operam o hardware.
            
Vou abrir um adendo para outros dois termos: 
1 – Sistemas de informação= Sistemas são conjuntos de componentes de softwares, que se relacionam 
e coletam, processam, armazenam e fornecem saída das informações com alguma finalidade específica, 
geralmente pra uso e interesse comercial.
2 - Um App é = Um software desenvolvido para uma única finalidade ou função, 
ou que se foca nos principais aspectos tecnológicos de dispositivos específicos.
            



        
Pegou a visão ?
 
São infinitas as definições, podemos usa-las exponencialmente!
Podemos trocar ou substituir os termos, manja ?! Mas o CONCEITO é o que representa a ideia. 
                
Por isso eu afirmei que EU (Allber), vejo mais relação entre esses conceitos do que diferença, 
então pra mim Programas & Softwares são estritamente ligados (mas isso não me torna o dono da razão, ok). 
Ah e se você analisar a fundo verá que todo Programa é um App ou vice-versa 
e que consequentemente app & programa são softwares... 
        
E abri aquele adendo também porque durante o desenvolvimento desse material me veio a mente 
que o sequencial e modulado Sistema de informação também pode ser composto 
por vários programas, aplicações e definitivamente esses complementos
se tornam softwares de soluções
